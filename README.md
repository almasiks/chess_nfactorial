# ♟️ Steppe Chess: Grandmaster Journey

**Steppe Chess** — это современная шахматная платформа с казахским колоритом, разработанная в рамках курса nFactorial. Мы объединили классическую игру с элементами RPG, системой обучения и глубокой культурной идентичностью.

> **Статус проекта:** Frontend Demo (интеграция с облачным бэкендом в процессе). 
> **Живая ссылка:** [https://chess-nfactorial-2glm.vercel.app/](https://chess-nfactorial-2glm.vercel.app/)

---

## 🌟 Ключевые особенности

### 🏹 Испытание Батыра (Daily Challenge)
Ежедневные шахматные задачи (мат в 3-4 хода). Решай их, чтобы заработать **Асыки** и повысить свой рейтинг. Встроена система подсказок от ИИ-советника.

### 🤖 AI Ústaz (Советник Хана)
Ваш персональный наставник. После каждой игры ИИ анализирует ваши ходы и дает советы на русском и казахском языках, помогая исправить тактические ошибки.

### 🛍️ Silk Road Bazaar (Магазин)
Тратьте заработанные в боях Асыки на уникальные товары:
- **Снаряжение:** Шахматные доски из ценных пород дерева, фигурки сарбазов.
- **Стиль:** Брендированные худи, футболки и "Рюкзак Кочевника".

### 📊 Система Каганатов (Leaderboard)
Соревнуйтесь за звание лучшего игрока в своем городе (Алматы), стране или во всем мире. 

### 📱 Полная адаптивность
Играйте с любого устройства. Интерфейс полностью перестраивается под мобильные телефоны, чтобы доска и управление всегда были под рукой.

## 🚀 Как пользоваться проектом

1. **Авторизация:** Нажмите "Войти" или "Регистрация". В текущей демо-версии данные сохраняются локально в вашем браузере.
2. **Игра:** Выберите режим "Играть с ботом" или "Игра вдвоем" (на одном экране). 
3. **Обучение:** Перейдите в раздел "Обучение", чтобы изучить теорию хода каждой фигуры: от Пешки до Короля.
4. **Задачи:** Зайдите в "Испытание батыра", чтобы решить ежедневную головоломку. Если возникнут сложности — нажмите "Подсказка".

---

## 🛠 Технический стек

- **Frontend:** React.js, Next.js, Tailwind CSS (для стилизации).
- **Game Engine:** `chess.js` (логика), `react-chessboard` (визуализация доски).
- **Animations:** Framer Motion.
- **Icons:** Lucide React.
- **State Management:** React Context API + LocalStorage (Demo Mode).

## 👨‍💻 Установка и запуск (Local)

Если вы хотите запустить проект локально:

```bash
# Клонировать репозиторий
git clone [https://github.com/ваш-логин/chess-nfactorial.git](https://github.com/ваш-логин/chess-nfactorial.git)

# Перейти в папку фронтенда
cd frontend

# Установить зависимости
npm install

# Запустить в режиме разработчика
npm run dev
